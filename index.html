The Tag <p>

The <p> tag is used to create a paragraph on a web page. For example, in order to create a paragraph, the text "Hello world!"

Example tag

Hello world!

<p>Hello world!</p>

The Tag <h1> - <h6>

The <h1> - <h6> tags are used to create different levels of headings on a web page, where <h1> is the most important and <h6> is the least important.

Example tag

Heading 5

<h5>Heading 5</h5>

The Tag <img>

The <img> tag is used to insert images into a web page. The src attribute specifies the path to the image file. In the alt attribute, you can specify a description of the image.

Example tag

<img alt="link/path to file" src="dog" width="600" height="600">

The Tag <iframe>

The <iframe> tag is used to distribute one web page to another location. It allows you to display the inner pages of another video frame's content. The attribute "src" specifies the URL of the page to be embedded. The "width" and "height" attributes specify the width and height of the frame in pixels, respectively.

Example tag

< iframe width="560" height="315" src="https://www.youtube.com/embed/TDoN3jkC-W4" title="YouTube video player" frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share" allowfullscreen> >/iframe>
